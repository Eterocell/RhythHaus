# i18n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Compose Multiplatform resource-backed English and Chinese UI strings for shared Compose screens.

**Architecture:** Create shared Compose string resources in `shared/src/commonMain/composeResources`. Replace hardcoded user-facing shared Compose strings with `stringResource(Res.string.*)` while keeping user media metadata and runtime values unchanged.

**Tech Stack:** Kotlin Multiplatform, Compose Multiplatform resources, existing shared Compose UI.

## Global Constraints

- Do not alter navigation, playback, scanner, Haze chrome, persistence, or platform entry-point behavior.
- Do not translate user media metadata.
- Keep English as default and add Simplified Chinese resources.
- Use existing Compose Multiplatform resources; do not add a new i18n dependency.
